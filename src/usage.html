Usage

Monitoring

  • Monitoring starts when you push this extension icon on the toolbar and open the monitoring screen. All tabs are monitored.
  • The browser will slow down while monitoring is in progress. If you do not need to monitor, close the monitoring tab or uncheck the capture checkbox.

Access Log

  • Displays a list of captured access logs on the monitoring screen.
  • Clicking a row in the list displays the HTTP Header details.
  • Right-clicking (or long-tapping) a row in the list opens a pop-up menu.

Pop-up Menu

  • The pop-up menu allows you to download and view the URL resource of the selected log.
  • If the size of the URL resource exceeds a certain size, you cannot view the URL resource. Change the URL resource size limit with the limit-size slider. The slider value takes effect from newly acquired access logs. Memory is consumed to hold the acquired resources for each access log.
